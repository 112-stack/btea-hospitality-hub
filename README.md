# BTEA Hospitality Hub

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Contributions welcome](https://img.shields.io/badge/contributions-welcome-brightgreen.svg)](CONTRIBUTING.md)

A modern React portal for bilingual hospitality operations, outlet administration, and email campaign management.

## Features

- English and Arabic outlet records with hospitality-specific fields.
- Email campaigns, reusable templates, teams, media assets, and administration.
- Responsive React interface with dark-mode-ready components.
- Form validation, charts, client state management, and session handling.
- Vite development workflow with linting, tests, and production builds.

## Getting started

```bash
npm install
npm run dev
```

Additional commands:

```bash
npm run lint
npm test
npm run build
```

Use synthetic data locally. Existing server endpoints in the UI are integration placeholders and may require a compatible backend.
